# csc205codeexamples - Lab Starter Code (C++ version)

This directory contains the C++ versions of the CSC205 lab starter code,
translated from the Java originals in
[`../../../java/src/labs`](../../../java/src/labs). See the links in your
Canvas course to ensure you are using the right starter code for your class.

Several of these are intentionally incomplete starter files (methods that
just `return 0`, empty `main()`s, etc.) — that is by design; the labs ask you
to fill them in. The translations keep those stubs as-is so you start from the
same place as the Java version.

Each lab folder has its own `Makefile`:

```
make        # compile the program
make run    # compile (if needed) and run it
make clean  # delete the compiled program
```

On Windows with MinGW, use `mingw32-make` instead of `make`.
